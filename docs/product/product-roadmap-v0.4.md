# BudgetApp — Product Roadmap v0.4

**Last updated:** 2026-03-06
**Previous release:** [v0.3](./product-roadmap-v0.3.md)

---

## Vision

v0.4 is a UX and UI quality pass across the entire application. Rather than shipping new features, this release focuses on reviewing and improving every major surface area — navigation, dashboard widgets, pages, and settings — one component at a time. Each area gets its own PRD defining the specific problems observed and the changes to ship.
